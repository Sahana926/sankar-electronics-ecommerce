# Homepage Hero Banner Image Prompt for Cursor

## Image Generation Prompt

Create a clean, modern, professional hero banner image for a website named **Shanker Electronics & Hardware**. 

### Visual Requirements:

**Color Scheme:**
- Primary: Blue and white color aesthetic
- Use soft gradients and subtle shadows
- Modern e-commerce style

**Content:**
- High-quality illustrations of electronic components
- Tools and hardware items
- Accessories arranged neatly
- Keep it minimal, balanced, and visually appealing

**Style:**
- Smooth glow effect
- Professional layout suitable for a homepage header
- Modern, clean design
- E-commerce aesthetic

**Important:**
- **DO NOT include any text in the image**
- Image should be suitable for a hero banner/header section
- Dimensions: 1920x600px (or similar wide banner format)
- High resolution, professional quality

### Suggested Prompt for Image Generation:

```
Create a clean, modern, professional hero banner image for an electronics and hardware e-commerce website. Use a blue-and-white color palette with soft gradients and subtle shadows. Include high-quality illustrations of electronic components (circuit boards, chips, resistors), tools (screwdrivers, pliers, wrenches), and hardware items (bolts, nuts, screws) arranged neatly in a balanced composition. Apply a smooth glow effect and maintain a minimal, professional layout suitable for a homepage header. Modern e-commerce style with clean aesthetics. No text should be included in the image. Dimensions: 1920x600px, high resolution.
```

### Alternative Shorter Prompt:

```
Modern e-commerce hero banner: blue-white gradient background, neatly arranged electronic components and hardware tools illustrations, soft glow effects, minimal professional layout, no text, 1920x600px
```

## How to Use in Cursor

1. Open Cursor's image generation feature
2. Copy and paste the prompt above
3. Generate the image
4. Save the image as `hero-banner.jpg` or `hero-banner.png`
5. Place it in the `public/` folder of your React project
6. Update the Home component to use the image

## Integration Steps

After generating the image:

1. Save the image to `public/hero-banner.jpg`
2. Update `src/pages/Home.jsx` to use the image as background
3. Adjust CSS to overlay text content on the image
